# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-07
### Added
- Initial stable release of the Axiom Real-Time Funding Prediction Engine.
- Comprehensive README with VodaPay commercial business case.
- Deterministic dependency tracking via `uv` (`pyproject.toml`).
- Automated testing and formatting hooks (`pre-commit`).
- Integration with Weights & Biases for experiment tracking.
- Core project structure mapped to the `actor_role_id` and `fund_order_status` schema.
